## Conseils pour les photos
- Format recommandé : JPG, qualité 85%
- Taille recommandée : 800×1000px minimum (ratio 3:4 pour les gâteaux)
- Rogner les photos pour centrer le gâteau (supprimer les arrière-plans encombrants)
- Outils gratuits : squoosh.app (compression), remove.bg (fond transparent)
